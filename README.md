# Django Ecommerce web application


# Description 
This is a **Ecommerce web application** built with **Django**, **HTML** and **CSS**.This project showcases the basic backend and frontend concepts in real-world applications.

# Features :
- User authentication(login,logout)
- Buy products
- add to cart
- role based login

# Technologies used :
- Python
- Django
- HTML
- CSS
