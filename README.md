# 🛒 Django E-Commerce Web Application
A full-featured **E-Commerce Web Application** built with **Django**, **HTML**, and **CSS**. This project demonstrates core full-stack development concepts including authentication, role-based access control, product management, shopping cart functionality, payment processing, and administrative analytics.


## 🚀 Features
### 👤 User Management
* User Registration & Login
* Secure Logout
* Role-Based Access Control
  * **Admin**: Manage products, view statistics, oversee platform activities
  * **Staff**: Add and manage products
  * **Customer**: Browse, search, and purchase products


### 🛍️ Shopping Experience
* Product Listing
* Product Search
* Category-Based Filtering
* Add to Cart
* Buy Now Functionality
* Order Placement & Management


### 📦 Product Management
* Add Products
* Edit Products
* Delete Products
* Product Categorization


### 💳 Payment Integration
* Razorpay Payment Gateway Integration
* Secure Checkout Process


### 📊 Admin Dashboard
* User management
* Staff management
* Product management

  
### 📞 Customer Support
* Contact Form for Customer Queries and Feedback


## 🛠️ Tech Stack
### Backend

### Frontend
* HTML

### Database
* SQLite (Default Django Database)


### Payment Gateway
* Razorpay


## ⚙️ Installation & Setup
### 1. Clone the Repository
```bash
git clone https://github.com/NandhuT17/django-webapp.git
```

### 2. Navigate to the Project Directory
```bash
cd django-webapp
```

### 3. Build the Project
```bash
bash build.sh
```

### 4.Start the Development Server
```bash
python manage.py runserver
```


## 🌐 Usage
After starting the server, open your browser and visit:
```
http://127.0.0.1:8000/
```
